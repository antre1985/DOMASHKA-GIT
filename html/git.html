Git

  • git init - Инициализирует репозиторий Git в текущем каталоге.
  • git add - Добавляет файлы в индекс Git для фиксации.
  • git commit - Создает фиксацию (снимок) изменений в репозитории Git.
  • git push - Отправляет изменения из локального репозитория в удаленный репозиторий.
  • git pull - Получает изменения из удаленного репозитория в локальный.
  • git status - Показывает текущее состояние репозитория (изменения, неотслеживаемые файлы и т.д.).
  • git branch - Создает, переключается между ветками и управляет ими.
  • git merge - Сливает изменения из одной ветки в другую.
  • git clone - Создает локальную копию удаленного репозитория Git.
  • git log - Выводит историю изменений в репозитории.
